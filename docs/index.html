CUSS2-ts Browser Examples

Welcome to the CUSS2-ts browser examples. These demonstrate how to integrate the CUSS2 TypeScript SDK in web applications.

Note: These examples connect to a local CUSS2 platform at http://localhost:22222. Make sure your CUSS2 platform is running before testing.

Available Examples

Basic Connection
A comprehensive example showing how to establish a connection to the CUSS2 platform, handle authentication, monitor connection events, and request state transitions. Uses the IIFE bundle loaded via a script tag.
ES Module Import
A minimal example demonstrating how to import CUSS2 as an ES module and establish a basic connection. Shows modern JavaScript module usage with import statements.
Interactive Tester
A comprehensive testing interface for the CUSS2 platform. Allows you to configure connection parameters, control application state transitions, and interact with various platform components through an intuitive UI.

Getting Started

To use CUSS2-ts in your own project, you can either:

  1. Use the IIFE bundle (for traditional script tags): <script src="https://unpkg.com/@cuss/cuss2-ts@latest/dist/cuss2.min.js"></script>
  2. Use the ES module (for modern JavaScript): import { Cuss2 } from "https://unpkg.com/@cuss/cuss2-ts@latest/dist/cuss2.esm.js";

Configuration

Both examples use the following default configuration:

  • Client ID: KAP
  • Client Secret: secret
  • WebSocket URL: http://localhost:22222

Update these values in the examples to match your CUSS2 platform configuration.
